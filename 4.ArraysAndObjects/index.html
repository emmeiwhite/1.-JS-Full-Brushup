Data Structures: Objects & Arrays

Numbers, booleans and strings are the atoms from which data structures are built. Many types of information require more than one atom though! Data Structure: A way to store information (values)

Objects

Objects allow us to group values - including other objects - to build more complex strucutures.

Arrays:

Suppose, we want to represent a collection of values say 2 , 3, 5, 7, & 11. One way would be to represent these with a string like this '2 3 5 7 11'. But the retrieval and future actions performed on it is going to be difficult.

JavaScript provides us the data type specifically for storing sequece of values . It is called an array .
