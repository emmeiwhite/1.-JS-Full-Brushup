Conditionals and Loops

for loop: Capable of running a piece of code repeatedly until either the condition becomes false , or a programmer explicitly uses a break statement

Summary

Program: Programs are made up of statements.

Expression: Fragment of code that produces a value .

Binding: Piece of data (values) under a name.

State: Bindings are used for tracking state in our program.

Environment: Collection of bindings and their values at a particular time.

Standard Bindings: JavaScript always puts the number of standard bindings in your environment. For example: document object, setTimeout, setInterval, fetch() etc etc etc

Function: Function is a special value in JavaScript that encapsulate piece of program. You can invoke them by writing functionName(argument1, argument2). Such a function call may result in an expression when the function returns the value.
